About This Project

Taka Bachai is a personal money management system developed as a course project. Built with Spring Boot on the backend and a clean vanilla HTML/CSS/JS frontend, it demonstrates full-stack web development with REST APIs, relational database design, and modern UI practices.

☕ Spring Boot 🗄️ H2 / Supabase 🌐 HTML / CSS / JS 📊 Chart.js
